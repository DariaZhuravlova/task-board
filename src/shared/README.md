# Shared Module

Общие утилиты, константы и API клиенты, используемые во всём приложении.

## Структура

```
shared/
├── api/               # API клиенты и запросы
│   └── api.ts
└── libs/
    └── constants/
        └── routes/
            └── routes.ts  # Константы маршрутов
```

## API (`api.ts`)

Централизованный клиент для API запросов. Содержит методы для:

- Получения задач
- Создания задач
- Обновления задач
- Удаления задач

## Константы

### Routes (`routes.ts`)

Содержит строковые константы для путей маршрутизации:

```typescript
export const routes = {
    home: "/",
    tasks: "/tasks",
    // ...
};
```

## Зависимости

- Fetch API для HTTP запросов
- TypeScript для типизации
