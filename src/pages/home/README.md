# Home Page Module

Модуль главной страницы приложения.

## Структура

```
pages/home/
├── index.ts           # Экспорт компонента
├── HomePage/
│   ├── HomePage.tsx   # Основной компонент страницы
│   └── HomePage.module.scss  # Стили компонента
```

## Компоненты

### HomePage

Главная страница приложения. Отображает приветствие и основную информацию.

## Стили

Использует SCSS модули для изоляции стилей. Основные классы:

- `.container` - основной контейнер
- `.title` - заголовок страницы
- `.content` - контент страницы

## Зависимости

- React
- SCSS modules для стилизации
