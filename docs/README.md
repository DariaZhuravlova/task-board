# Task Board - Документация

## Обзор проекта

Task Board - это веб-приложение для управления задачами, построенное с использованием React, TypeScript, Redux Toolkit, Tailwind CSS и shadcn/ui.

## Структура проекта

```
src/
├── app/                 # Основное приложение
│   ├── config/          # Конфигурация маршрутов и store
│   ├── providers/       # Провайдеры (Redux)
│   ├── styles/          # Глобальные стили
│   └── components/      # UI компоненты
├── pages/               # Страницы приложения
├── shared/              # Общие утилиты и API
└── assets/              # Статические ресурсы
```

## Технологии

- **Frontend**: React 19, TypeScript
- **State Management**: Redux Toolkit
- **Styling**: Tailwind CSS, SCSS modules
- **UI Components**: shadcn/ui
- **Build Tool**: Vite
- **Routing**: React Router
